# marko-redux-example

[![Greenkeeper badge](https://badges.greenkeeper.io/charlieduong94/marko-redux-example.svg)](https://greenkeeper.io/)

This is a simple example of how to use
[marko-redux](https://github.com/charlieduong94/marko-redux).

## Installation

```sh
npm install

# or
yarn install
```

## Starting the app

```sh
npm start

# or
yarn start
```
